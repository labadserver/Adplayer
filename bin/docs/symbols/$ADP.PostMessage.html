Class Detail
$ADP.PostMessage()
Static class for PostMessage communication. Implementation only supports postMessage for "modern-based" browsers. No alternate communication ability is setup for older browsers at this time using a data funnel solution similar to postMessage. Please see the apstub.html for other iframe communication techniques.

Author: christopher.sancho@adtech.com.
