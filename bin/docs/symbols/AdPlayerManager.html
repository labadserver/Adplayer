Class Detail
AdPlayerManager()
Globally Manages all created AdPlayer instances. AdPlayerManager is a singleton class and ensures it is the only available AdPlayerManager throughout an ad delivery flow.

Author: christopher.sancho@adtech.com.

Method Detail
addAdPlayer(adPlayer)
Adds an AdPlayer instance to the management list. When a new AdPlayer instance is created, it is automatically passed to this method. Immediately following, all call-backs, registerd through AdPlayerManger.register(adPlayer) are dispatched and passed with the newly created AdPlayer. 
// Add an AdPlayer instance to the manager.
var adPlayer = new AdPlayer(document.getElementById('myTagDivContainer'));
AdPlayerManager.addAdPlayer(adPlayer);
Parameters:
{adplayer} adPlayer
AdPlayer instance to add to management list.

						Registers a function that will be called when an AdPlayer instance

  adPlayer.addPrivacyInfo('AD_SERVER', 'My message goes here.', 'http://adplayer.aboutthisad.com');
