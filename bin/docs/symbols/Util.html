Method Detail
jsonParse(txt, reviver, rdyFn)
Method first checks if JSON is natively available in the browser. If not, it will attempt to load an external JSON framework script. Finally it will initialize a callback when a successful parse is complete. 
var str = '{"hello":"world"}';
Util.jsonParse(str, null, 
  function(json) {
    Util.log(json.hello);
  }
);
Parameters:
{string} txt
A valid JSON string to parse.
{function} reviver
Function called for every key value from parsed result.
{function} rdyFn
Callback function called and passed the parsed JSON object.

loadScript(objId, objReturnFn, scriptSrc, callback)
Loads an external script & executes a callback function when an object, located in the external script & is returned in a testFn, is valid. 
Util.loadScript('ExtScriptObj', function(){return ExtScriptObj;}, 'http://the.script.url/extobj.js', 
  function(){
     Util.log('External script is done loading.');
  }
);
Parameters:
{string} objId
Id used to identify script. Note: Implemented to avoid use of eval();
{function} objReturnFn
Function that returns object to check against. Note: Implemented to avoid use of eval();
{url} scriptSrc
The url of the script to load.
{string} callback
The handler to be executed when script load is complete.

ready(testFn, context, readyFn, readyParams, errorFn, errorParams)
Executes a callback function when an object, being returned in a testFn, is valid. 
function onReady(msg, num) {
  Util.log(msg + ':' + num);
}

function onError(msg) {
  Util.log(msg);
}

Util.ready(function(){return testObj;}, this, onReady, ['Hello World!', 100], onError, ['Error...']);
Parameters:
{function} testFn
Function that returns object to test against. Note: Implemented to avoid use of eval();
{object} context
Context of where readyFn & errorFn are located.
{function} readyFn
Function called when object being tested returns true.
{array} readyParams
Array of parameters to pass at the time readyFn is called.
{function} errorFn
Function called when object being tested can not be found. Method times out at a count of 100 at 100ms intervals.
{Array} errorParams
Array of parameters to pass at the time errorFn is called.
