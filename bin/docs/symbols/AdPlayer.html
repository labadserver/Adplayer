addEventListener(adEvent, callback)
Adds a callback function to an AdEvent flow. Callback handler function returns an AdEvent object instance, which contains the following:
  • data - Object containing any data passed at the time of dispatch.
  • type - The AdEvent type passed at the time if dispatch.
  • target - Reference to the parent AdPlayer within the setup hierarchy.
  • currentTarget - Reference to the current AdPlayer from where the dispatch orignated.
function trackEventHandler(adEvent) {
 Util.log(adEvent.type() + ' has been dispatched');
 Util.log(adEvent.data().message);
}
adPlayer.addEventListener(AdEvent.TRACK, trackEventHandler);

var data = new Object();
data.message = "Hello World!";
adPlayer.track(new AdEvent(AdEvent.TRACK, data), trackEventHandler);
Parameters:
adEvent
{AdEvent} The AdEvent string to listen to.
callback
{Function} The callback handler to call when an AdEvent is dispatched.

addTrackingPixel(adEvent, url, repeat)
A convenience function that adds a pixel URL to an AdEvent flow.

adPlayer.addTrackingPixel(AdEvent.TRACK, 'http://my.pixel.url');

removeTrackingPixel(adEvent, url)
Removes a matching url or callback associated with an AdEvent and registered using addTrackingPixel() method.
  • If a callback and a url are both defined, method will remove the AdEvent associated with the callback.
  • If only a callback is defined method will remove the AdEvent associated with the callback.
  • If only a url is defined, method will remove all AdEvent callbacks containing a matching url property.
adPlayer.addTrackingPixel(AdEvent.TRACK, 'http://my.pixel.url');
adPlayer.addEventListener(AdEvent.TRACK, removePixels);

function removePixels(adEvent) {
 // Removes url from TRACK event flow.
 adPlayer.removeTrackingPixel(AdEvent.TRACK, 'http://my.pixel.url');
 
}
Parameters:
adEvent
{AdEvent} The AdEvent to listen to.
url
{String - URL} Optional - URL of pixel to call when associated AdEvent is dispatched.

showPrivacyInfo()
Convenience function that creates a layer that displays the privacy info added to privacyInfoList.
AdEvent.PRIVACY_CLOSE is dispatched when method is called.
Layer should be styled to preference using css.
  • .privacyPanel
  • .privacyPanel ul
  • .privacyPanel ul li
  • .privacyPanel button
Placement of the privacy panel can be modified by targeting the following css classes:
  • .top-right
  • .top-right-out
  • .top-left
  • .top-left-out
  • .bottom-right
  • .bottom-right-out
  • .bottom-left
